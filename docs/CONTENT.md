# Road Beans Content Style Guide

Source of truth for in-app copy. Every downstream feature spec references this file rather than re-deriving rules.

## Positioning

Road Beans is a serious tool for people who care about coffee. Premium specialty-coffee aesthetic. Not gamified. Not playful. Quiet, confident, useful.

## Voice Rules

1. Sentence case. Proper nouns and the wordmark are the exceptions.
2. Imperative for actions. Declarative for state. Never interrogative.
3. Numbers round humanely.
4. Do not address the reader as "you" in headlines. Use it sparingly in body.
5. No exclamation marks.
6. No emoji.
7. User content renders verbatim.

## Banned Patterns

- "Welcome", "Hi there", "Hello", "Yay", "Oops", "Whoops", "Uh oh"
- "Let's" and "We"
- "Tap to" hints inside button labels
- "Please" and "Sorry"
- Multi-sentence button labels
- Exclamation marks
- Emoji

## Common Moments

| Moment | Approved | Banned |
|---|---|---|
| Empty list | "No visits yet. Add one when something is worth remembering." | "You haven't added any visits yet" |
| Save success | "Saved." | "Visit saved successfully" |
| Error | "Couldn't save. Check your connection and try again." | "Oops. Something went wrong." |
| Confirm destructive | "Delete this visit?" + "Delete" / "Cancel" | "Are you sure you want to delete this visit?" |
| Onboarding | "A field guide to your coffee." | "Welcome to Road Beans" |
| Permission ask | "Road Beans needs location access to show nearby places." | "We'd love access to your location." |
| Search empty | "No matches." | "Sorry, no results found." |

## Numbers, Dates, Times

- Counts: integer always. "12 visits," "3 places."
- Ratings: one decimal. Use the bean mark for rating visuals.
- Dates: "Apr 27" for current year, "Apr 27, 2024" for other years.
- Times: use locale-aware time formatting.
- Relative dates are acceptable for recent community feed moments; after 2 weeks, use absolute dates.

## User Content

Place names, drink names, and tags render verbatim. Do not auto-correct, re-capitalize, or rewrite them.

## Tagline

The official tagline is: **A field guide to your coffee.**

Use it only on App Store hero materials and the onboarding splash.
